## 《算法经典入门 紫书》刘汝佳
习题开始从第三章开始一题一题训练过去，语言为c++
每题的命名格式为 chapter-nums-[ac]-uva__-[acok].cpp 
* [ac]表示看别人代码跑通的，

* [acok]表示自己做题完成的，

* chapter为该本书的章节，nums表示该按顺序出现的uva题目，

* uva__ 表示具体uva题目序列号
